# 百度 学习笔记

## 前言
百度，2000年诞生于北京中关村，得名于“众里寻他千百度”的千古佳句，以“用科技让复杂的世界更简单”为使命，从中文搜索引擎起步，成长为全球四大AI公司之一。二十余年深耕中，百度始终以技术创新为信仰，构建起“芯片-框架-模型-应用”的全栈AI布局，文心大模型、昆仑芯、萝卜快跑等核心成果持续引领行业，近十年累计研发投入超1800亿元，AI专利申请量稳居中国第一。如今，百度以“夯实搜索核心、开拓AI创新”为战略，形成移动生态、智能云、智能交通等多引擎增长格局，让搜索从“找到信息”升级为“交付结果”，让AI深度融入千行百业与日常生活，向着“成为最懂用户，并能帮助人们成长的全球顶级高科技公司”稳步迈进。
